start MODE

Start a mode, verifier, symbolic and concrete simulator require a valid document.

import ( document | templates | queries ) FILENAME

Import a document or only some of its queries or templates.

export ( document | trace  | queries ) FILENAME

export a document, a trace or a query list to the specified file.

unset [ document | queries | REF ]

Clear the current document, list of queries or members of a reference.

set (template name (parameter) | query formula ("comment")? | ref = value
Set the value of a property, either on the command line or through a dedicated screen.
 
show REF

Show the statement attached to a reference.

select REF | STATE | TRANSITION

Select either the statement attached to a reference, a state or a transition.

unselect [ REF ]

unselect the statement attached to a reference.


check QUERRYREF

Check a query, all selected ones if no reference is given.

help ( MODE | OPERATION )

Show the help about a mode or an operation.


exit

Terminate the uppaal command line.

ref : name ([name (,name)?])? (. name)? | query[number]
reference of a node, of a property or of a query
